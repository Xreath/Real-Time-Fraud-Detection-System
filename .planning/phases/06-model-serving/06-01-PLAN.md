   - CATEGORICAL_COLUMNS = ["merchant_category", "card_type", "country"]

2. **class FraudPyfunc(mlflow.pyfunc.PythonModel)** with:

   `load_context(self, context)`:
   - Load `self.scaler = joblib.load(context.artifacts["scaler"])`
   - Load `self.label_encoders = joblib.load(context.artifacts["label_encoders"])`
   - Load `self.feature_names = joblib.load(context.artifacts["feature_names"])`
   - Load `self.threshold = float(os.getenv("FRAUD_THRESHOLD", "0.5"))` (per D-03)
   - Load the LightGBM model: `self.model = joblib.load(context.artifacts["lgbm_model"])` — the model file downloaded from MLflow is a joblib-serialized LGBMClassifier (use mlflow.artifacts.download_artifacts to get it as a local directory, then load model.pkl or use mlflow.sklearn.load_model from the local path)

   `predict(self, context, model_input: pd.DataFrame, params=None)` (per D-02):
   - `df = model_input.copy()`
   - Categorical encoding: for each col in CATEGORICAL_COLUMNS, if col in df.columns, apply `le = self.label_encoders[col]`, then `df[col] = df[col].astype(str).apply(lambda x: le.transform([x])[0] if x in le.classes_ else 0)` — exactly as FraudScorer.preprocess()
   - Prior-transaction fields: for col in ["prev_latitude", "prev_longitude", "prev_timestamp"], if col not in df.columns, set `df[col] = 0.0` (Claude's discretion: optional with zero fallback)
   - Feature selection: `X = df[self.feature_names].fillna(0).values`
   - Scaling: `X = self.scaler.transform(X)`
   - Scoring: `probs = self.model.predict_proba(X)[:, 1]`
   - Return: `pd.DataFrame({"fraud_score": probs.astype(float), "fraud_prediction": (probs >= self.threshold).astype(int)})`

3. **def register_pyfunc_model()** function:
   - Set MLFLOW_TRACKING_URI from config: `mlflow.set_tracking_uri(MLFLOW_TRACKING_URI)`
   - Set MinIO env vars: `os.environ.setdefault("MLFLOW_S3_ENDPOINT_URL", "http://localhost:9000")`, same for AWS_ACCESS_KEY_ID=minioadmin, AWS_SECRET_ACCESS_KEY=minioadmin
   - Resolve current champion: `client = mlflow.tracking.MlflowClient()`, `mv = client.get_model_version_by_alias("fraud-detection-model", "champion")`, `champion_run_id = mv.run_id`
   - Download champion LightGBM model to temp dir: use `tempfile.mkdtemp()`, then `mlflow.artifacts.download_artifacts(run_id=champion_run_id, artifact_path="model", dst_path=tmpdir)` to get the model directory locally — the local path will be `{tmpdir}/model`
   - Call `mlflow.set_experiment("fraud-detection")`
   - Call `mlflow.pyfunc.log_model(artifact_path="model", python_model=FraudPyfunc(), artifacts={"scaler": "data/artifacts/scaler.joblib", "label_encoders": "data/artifacts/label_encoders.joblib", "feature_names": "data/artifacts/feature_names.joblib", "lgbm_model": f"{tmpdir}/model"}, pip_requirements=["mlflow==2.12.2", "lightgbm>=4.0.0", "scikit-learn>=1.4.0", "joblib>=1.3.0", "pandas>=2.2.0", "numpy>=1.26.0", "boto3>=1.34.0"], registered_model_name="fraud-detection-model")` (per D-01, D-04)
   - Get new version number from the returned ModelInfo: `result.registered_model_version`
   - Move champion alias: `client.set_registered_model_alias(name="fraud-detection-model", alias="champion", version=new_version)` (per D-04)
   - Print confirmation with version number

4. **if __name__ == "__main__"** block: call `register_pyfunc_model()`
